# AGENTS.md Design

## Goal

Add a concise root AGENTS.md for contributors working inside this repository.

## Approach

Use a single root file with only repository-specific instructions that help coding agents work safely in this codebase without duplicating the full README or existing skill files.

## Included Sections

- Scope and precedence
- Project shape
- Editing rules
- Unity workflow
- Verification
- Documentation expectations

## Exclusions

- No downstream Unity project usage guide
- No duplicated tool reference from README
- No broad contribution policy already covered elsewhere
